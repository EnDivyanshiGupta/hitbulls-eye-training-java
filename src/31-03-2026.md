# Power of Two Solution

This document contains a Java implementation to determine if a given integer is a power of two using bitwise operations.

## Implementation

```java
class Solution {
    /**
     * Checks if a given integer is a power of two.
     * * @param n The integer to check.
     * @return true if n is a power of two, false otherwise.
     */
    public boolean isPowerOfTwo(int n) {
        // Powers of two must be positive
        if (n <= 0) return false;
        
        // A power of two in binary has exactly one bit set.
        // Subtracting 1 flips all bits after the rightmost set bit.
        // Using bitwise AND results in 0 if only one bit was set.
        return (n & (n - 1)) == 0;
    }
}

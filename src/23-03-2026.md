# LeetCode 509 - Fibonacci Number

## Problem
Given an integer `n`, return the `n`-th Fibonacci number.

Fibonacci sequence:
F(0) = 0  
F(1) = 1  
F(n) = F(n-1) + F(n-2)

---

## Java Solution (Optimal - Iterative)

```java
class Solution {
    public int fib(int n) {
        if (n <= 1) return n;

        int prev = 0, curr = 1;

        for (int i = 2; i <= n; i++) {
            int next = prev + curr;
            prev = curr;
            curr = next;
        }

        return curr;
    }
}

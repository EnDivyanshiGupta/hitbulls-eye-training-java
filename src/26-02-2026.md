# Copy List with Random Pointer

## Java Solution

```java
class Solution {
    public Node copyRandomList(Node head) {
        Node Head = null;
        Node Tail = null;
        Node curr = head;

        // Step 1: Create a copy of the linked list (only next pointers)
        while(curr != null){
            Node newNode = new Node(curr.val);

            if(Head == null){
                Head = newNode;
                Tail = newNode;
            }
            else{
                Tail.next = newNode;
                Tail = newNode;
            }

            curr = curr.next;
        }

        // Step 2: Assign random pointers
        curr = head;
        Node ptr = Head;

        while(curr != null){
            int idx = 1;
            Node temp = head;

            // Find index of random node
            while(temp != curr.random){
                temp = temp.next;
                idx++;
            }

            // Move to same index in copied list
            temp = Head;
            while(idx >= 2){
                temp = temp.next;
                idx = idx - 1;
            }

            ptr.random = temp;

            curr = curr.next;
            ptr = ptr.next;
        }

        return Head;
    }
}
```

## Notes
- First loop creates a deep copy of nodes using `next` pointer.
- Second loop assigns `random` pointers by calculating index.

- Space Complexity: O(n)
